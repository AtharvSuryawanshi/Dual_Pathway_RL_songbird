How does the brain learn complex sensorimotor behaviors? We explore song learning in zebra finches and reveal how a dual pathway architecture implements the exploration-exploitation tradeoff, offering insights into how neural circuits support learning and adaptive behavior.

This is a repository to store the codes of a rate coding of a Dual Pathway Learning model in Song Birds.

The intermediatary report of the work done until July 2024 is attached as `internship_report.pdf`. 
Currently the paper is undergoing its first draft towards publication.
